# هماهنگ — افزونه‌ی مدیریت پروژه و تیم (CPTT)

نسخه: **5.4.4**

افزونه‌ی وردپرسی مدیریت پروژه‌ی مشتری-محور با Stepper، چک‌لیست، تسک سمت مشتری، گزارش نهایی، حساب و کتاب، پنل کارشناس، اتصال به ووکامرس، پنل کامل ربات بله، و سیستم ورود/ثبت‌نام با OTP بله.

## ✨ تغییرات نسخه‌ی 5.4.4

### UI / UX
- 🎨 **دکمه‌های شناور موبایل** همتراز با دکمه‌ی منو در پایین صفحه قرار گرفتند (Back-to-top چپ، Save وسط، Menu راست) و هنگام بازشدن مودال موبایل **مخفی می‌شوند**.
- 📐 **دکمه‌های کارت پروژه** (مدیریت پروژه / پیش‌فاکتور / گزارش) فشرده شدند: در دسکتاپ همه در یک ردیف، در موبایل «مدیریت پروژه + پیش‌فاکتور» در یک ردیف و «گزارش نهایی» فول‌ویدث در ردیف بعدی.

### ربات بله — بازنویسی کامل
- 🎛 **همه‌ی کامندها به دکمه‌های شیشه‌ای تبدیل شدند** (ایجاد پروژه، broadcast، تنظیمات اعلان، …). دیگر نیاز به تایپ کامند نیست.
- ✏ **edit-in-place**: کلیک روی دکمه پیام قبلی را به‌جای ساخت پیام جدید ویرایش می‌کند.
- 🧙 **Wizard چندمرحله‌ای ایجاد پروژه**: عنوان → شماره مشتری → انتخاب تمپلیت → هزینه → کارشناس → تأیید (با state و TTL).
- 📁 **پنل کارشناس کامل در ربات**: پروژه‌های فعال، مشاهده مراحل، تغییر وضعیت، چک‌لیست (toggle با یک کلیک)، چت پروژه (متن+فایل).
- 📝 **پاسخ مشتری به تسک در ربات**: با دکمه‌ی «پاسخ»، متن و/یا فایل ارسال کنید — فایل خودکار در media library آپلود می‌شود.
- 💰 **موجودی و تاریخچه تسویه** برای کارشناس در ربات.
- 🔕 **Mute به‌ازای هر پروژه**: ۱ ساعت / ۱ روز / تا اطلاع ثانوی.
- 📊 **داشبورد مدیر در ربات**: آمار کل، وضعیت امروز، کارهای معوق، تسویه‌های در انتظار.
- ⏰ **هشدار هوشمند Deadline + کرون گزارش روزانه/هفتگی**: هر روز صبح خلاصه‌ی کارهای امروز و معوق برای مدیر و کارشناس.
- 🎉 **پیام تکمیل پروژه** با دکمه‌ی «مشاهده گزارش نهایی» مستقیماً به مشتری و ادمین.
- 🎨 **بازنویسی همه‌ی متون اعلان‌ها** با ایموجی و قالب‌بندی Markdown زیباتر.

### سیستم ورود اختصاصی با OTP بله
- 🔐 صفحه‌ی **`/cptt-login`** با طراحی مدرن، RTL، ریسپانسیو، با گرادیان متحرک.
- ورود/ثبت‌نام با شماره موبایل + دریافت **کد ۵ رقمی در ربات بله**.
- اگر کاربر به بله متصل نیست، راهنمایی گام‌به‌گام برای اتصال نشان داده می‌شود.
- Rate-limit ۶۰ ثانیه‌ای + Brute-force protection (حداکثر ۵ تلاش برای هر کد).
- گزینه‌ی **«جایگزین وردپرس»** برای ریدایرکت `wp-login.php` به فرم اختصاصی.
- در صورت ثبت‌نام جدید، کاربر خودکار با نقش `customer` و username = شماره موبایل ساخته می‌شود.

## ساختار
- `client-project-tracker.php` — بوت‌استرپ + activation hooks
- `includes/`
  - `class-cptt-core.php` — CPTها، نقش کارشناس، توابع جلالی
  - `class-cptt-admin.php` — متاباکس‌ها، داشبورد، حساب و کتاب + تسویه مرحله‌ای
  - `class-cptt-expert.php` — پنل کارشناس، نوتیفیکیشن‌ها، چت
  - `class-cptt-frontend.php` — شورتکد + endpoint ووکامرس
  - `class-cptt-report.php` — گزارش نهایی پروژه
  - `class-cptt-settings.php` — تنظیمات (شامل تب بله + OTP)
  - `class-cptt-sms.php` — پیامک webhook + کرون
  - `class-cptt-woocommerce.php` — ساخت خودکار پروژه از سفارش
  - `class-cptt-analytics.php` — آمار و گزارش
  - `class-cptt-bale.php` — **ربات بله v5.4.4 (state machine + wizard + کرون)**
  - `class-cptt-auth.php` — **سیستم ورود OTP با بله (v5.4.4 جدید)**
- `assets/css/`, `assets/js/`, `assets/fonts/`

## نصب
1. فایل زیپ را در `wp-admin → افزونه‌ها → افزودن → بارگذاری` آپلود کنید.
2. فعال‌سازی → از منوی «پروژه‌های مشتری» → «تنظیمات» تب «ربات بله» شروع کنید.
3. توکن ربات بله و آی‌دی ادمین را وارد کنید.
4. در صورت نیاز، گزینه‌ی «فعال‌سازی ورود اختصاصی» و «جایگزین wp-login» را روشن کنید.
5. یک‌بار از تنظیمات وردپرس → پیوندهای یکتا → «ذخیره» را بزنید تا rewriteها رفرش شوند.
